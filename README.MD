# Setup

### 1. Pull the repo
### 2. Setup

from root folder run:
Windows
```shell
python -m venv venv
venv\Scripts\activate
python -m pip install -r ./requirements.txt
```

Linux
```sh
python3 -m venv venv
source venv/bin/activate
python3 -m pip install -r ./requirements.txt
```

### 3. Run

Windows:
```sh
python main.py
```
Linux:
```sh
python3 main.py
```
